# ReachMobi Portfolio Analysis

A senior-data-analyst proof-of-work: end-to-end LTV, CAC, and payback analysis
across a synthetic 4-app portfolio modeled after ReachMobi's studio categories
(launcher, email, messaging, personalization).

**Status:** in progress — Hour 1 of 5.
